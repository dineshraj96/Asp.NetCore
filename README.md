# Asp.NetCore
Sample Basic Asp.Net Core Project


It has only two pages, one for authentication and the other is to add users. 
Following are the modules presented inside the project as well i explored:
Asp.net core MVC- WebApi,
Asp.net core project structure,
Static files,
Configuration and startup File,
Entity Framework Core - Code First, DB First,
Referencing .Net Framework DLL.

Note: Application has no logic. It has been created to explore and experience the features. It cannot be executed since the Database connection is local.
